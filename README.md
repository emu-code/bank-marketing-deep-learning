# Bank Marketing Deep Learning Project

## Overview
A deep learning model built with TensorFlow/Keras to predict 
whether a customer will subscribe to a bank term deposit.

## Dataset
Bank Marketing Dataset — 11,162 samples, 17 features

## Model Results
- Test Accuracy: 81%
- F1 Score: 0.81

## Libraries Used
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib / Seaborn
